# Мій перший веб-сайт (Лабораторна робота №1)

Цей проект є базовим шаблоном веб-сторінки, створеним для вивчення структури HTML, CSS та JS.

## Структура проекту
- `index.html` — головна сторінка.
- `css/style.css` — стилі оформлення.
- `js/script.js` — скрипти для інтерактивності.
- `pages/about.html` — сторінка "Про нас".

## Як запустити проект
1. Склонуйте репозиторій:
   git clone https://github.com/ivannaspas11/my-first-website.git
2. Відкрийте файл `index.html` у будь-якому браузері.
3. Або просто відкрийте проект у [CodeSandbox](https://codesandbox.io).

## Технології
- HTML5
- CSS3
- JavaScript
